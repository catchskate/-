基于Pix2Text+PyQT5完成的可视化数学公式识别提取工具，可直接提取为LaxTex、MathML格式。  
也可仅识别文本内容（准确率低）  
--------------开发功能--------------  
1.用户自行上传图片进行识别  
2.使用内置截图功能进行截图识别  
3.支持仅公式、仅文字、公式+文字三种识别
